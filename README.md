## :rocket: Tecnologias Utilizadas

- Node.js
- MongoDB
- Mongoose
- Date-fns

<br>

### :dvd: Rodando o Projeto

Clone o repositório, entre no diretório "backend" e execute os comandos.
:loudspeaker: É necessário ter o mongoDB instalado.
```javascript
yarn install 

yarn dev 
```
